## 🌐 Specialized Search Engines

I learned about specific search engines that focus on particular types of results.

- **Shodan**  
  → A search engine that finds devices connected to the internet, such as servers, network equipment, and information about them, such as versions and exposure.

- **Censys**  
  → It's similar to Shodan, but it's geared towards hosts, websites, certificates, and other things  

- **VirusTotal**  
  → It's an online site that allows you to scan files, URLs, and hashes for malware.  

- **Have I Been Pwned**  
  → Checks if an email was exposed in data breaches. Since many users use the same password for multiple accounts, if one email account is compromised, it could expose other accounts.

---

## 👾 Vulnerabilities and Exploits

In this module, I learned about some applications useful for looking up exploits and existing vulnerabilities.

- **CVE (Common Vulnerabilities and Exposures)**  
  → Provides vulnerability entries where each vulnerability has a unique identifier, so that any IT professional can reference the same vulnerability.
- **Exploit Database**  
  → As the name suggests, it is a database with exploits and vulnerabilities tested and discovered by multiple authors, making it a great tool to help pentesters and red teams.
- **GitHub**  
  → On GitHub, there are several tools that help with vulnerability searches. A practical example would be to search for a vulnerability in CVE and paste the identifier into GitHub to find a specific tool to use.
